# Mockups moviles profesionales - AlertaBombero PWA

Estas imagenes son prototipos visuales generados con IA para guiar la version movil de la PWA. Estan basadas en el Figma de referencia y en la especificacion funcional del proyecto.

> Nota: los textos dentro de imagenes generadas por IA pueden tener pequenas imperfecciones. Para implementacion, recrear la UI en codigo con textos exactos.
